Модуль BestBuyContentInsert

Вставляет контент в текст статьи, сгенерированный в кастомном шаблоне из полей-аттрибутов статьи (Free Text Fields)


1. Активация вставки контента.
Отметить чекбокс "Content insert".
При отключении чекбокса, при выводе статьи, токен для замены из текста статьи удаляется (заменяется на пустую строку)

2. Добавление токена для замены.

В поле Token ввести наименование токена (например: CONTENT_INSERT)
В тексте статьи, в нужном месте, вставить токен, оберныутф в двойные квадратные скобки (например: [[CONTENT_INSERT]])

3. Поля-атрибуты, передаваемые в шаблон.
В поле Attributes ввести имена полей-атрибутов, которые необходимо использовать в шаблоне генерации контента для вставки.
В шаблоне, поля доступны как свойства smarty-объекта sVars.
Например значение поля banner_logo в шаблоне доступно так: {sVars.banner_logo}

4.Имя файла-шаблона из папки модуля: BestBuyContentInsert/Resources/views

5.Поля-атрибуты, содержащие контент, используемый для генерации вставляемого контента, могут быть произвольные.
Главное, указать их в поле Attributes и задействовать в smarty-шаблоне.
